# dce-plugin-sdk-java

DCE plugin SDK for Java.


## Requirements

- Java 1.8

## Installation

### Install from Maven:

Put this in your `pom.xml`:

```xml
<dependency>
  <groupId>io.daocloud.dce</groupId>
  <artifactId>dce-plugin-sdk</artifactId>
  <version>0.1</version>
</dependency>
```
